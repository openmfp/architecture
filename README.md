# openMFP - Architecture

## Description
This document contains Requests for Comments (RFC) and Architecture Decision Records (ADRs) to document the architecture and decisions made in the openMFP project.

This evolving library of records will help all contributors to understand the reasoning behind the architecture and design decisions made in the project.
